# 常见 Conda 命令

## 初始化

```bash
conda init
```

## 创建虚拟环境

```bash
conda create -n env_name python=3.7
```

## 列出所有虚拟环境

```bash
conda env list
```

## 删除虚拟环境

```bash
conda env remove --name env_name
```

## 删除索引缓存、锁定文件、未使用过的包和tar包

```bash
conda clean --all
```

## 查看当前 Conda 信息

```bash
conda info
```
